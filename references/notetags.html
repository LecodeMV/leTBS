# All Notetags

Here you can find and sort all the possible notetags of LeTBS.
The icons displayed show how compatible the tags are with the database items:

  • : Compatible with everything
  • : Compatible with actors and classes
  • : Compatible with enemies
  • : Compatible with equipment
  • : Compatible with states
  • : Compatible with skills and items
  • : Compatible with events

Important Note

Remember this priority list when the same tag is assigned on different objects for the same battler:
State > Equipment > Item in usage (Item or Skill) > Class > Battler
