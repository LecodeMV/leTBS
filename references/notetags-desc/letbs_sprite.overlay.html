# letbs_sprite.overlay

This is a wrapper tag that help you organize all the configuration related to sprite overlays.

See Sprite Overlays.

# Usage

<letbs_sprite>

<overlay>
...configuration here
</overlay>

</letbs_sprite>

# Examples

<letbs_sprite>

<overlay>
chara(idle): $hat
</overlay>

</letbs_sprite>
